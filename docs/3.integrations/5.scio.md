# scio Integration

[scio](https://github.com/zoobzio/scio) is a data catalog that tracks registered data stores. Sum automatically populates the catalog when you use data helpers.

## What scio Provides

- Registry of data stores (databases, KV stores, object storage)
- Health check aggregation
- Metrics collection points
- Service discovery for data infrastructure

## The Integration

Sum's `Service` embeds a scio catalog:

```go
svc := sum.New(cfg)
catalog := svc.Catalog() // *scio.Scio
```

Data helpers automatically register with this catalog:

| Helper | Catalog Method | URI Pattern |
|--------|----------------|-------------|
| `Database` | `RegisterDatabase` | `db://tablename` |
| `Store` | `RegisterStore` | `kv://storename` |
| `Bucket` | `RegisterBucket` | `bcs://bucketname` |

## Automatic Registration

When you use a data helper, it registers with the catalog:

```go
err := sum.Database[UserRepo, User](k, db, "users", "id", renderer, factory)
// Catalog now has: db://users
```

The catalog tracks the store's atomic for health and metrics.

## Accessing the Catalog

Get the catalog from the service:

```go
svc := sum.New(cfg)
catalog := svc.Catalog()
```

## Manual Registration

For stores not created via helpers, register manually:

```go
// Create store directly
gdb, _ := grub.NewDatabase[User](db, "users", "id", renderer)

// Register with catalog
catalog := svc.Catalog()
err := catalog.RegisterDatabase("db://users", gdb.Atomic())
```

## Use Cases

### Health Checks

Aggregate health across all registered stores:

```go
catalog := svc.Catalog()
// Check health of all registered stores
```

### Service Discovery

List available data stores:

```go
catalog := svc.Catalog()
// Enumerate db://, kv://, bcs:// entries
```

### Metrics

Collect metrics from store atomics:

```go
catalog := svc.Catalog()
// Access atomic metrics for each store
```

## Direct scio Usage

For advanced catalog operations:

```go
import "github.com/zoobzio/scio"

catalog := scio.New()

// Manual registration
catalog.RegisterDatabase("db://custom", atomic)
catalog.RegisterStore("kv://custom", atomic)
catalog.RegisterBucket("bcs://custom", atomic)
```
